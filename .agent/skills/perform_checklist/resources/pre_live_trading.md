# Pre-Live Trading Checklist

⚠️ **CRITICAL**: Complete ALL items before enabling live trading with real funds.

## Paper Trading Validation
- [ ] Paper traded for minimum 2 weeks
- [ ] Minimum 50 trades executed in paper mode
- [ ] Positive or expected PnL in paper mode
- [ ] No unexpected errors or crashes
- [ ] Position reconciliation tested (restart mid-position)

## Risk Controls
- [ ] Maximum position size configured
- [ ] Stop loss functioning correctly
- [ ] Circuit breaker tested
- [ ] Daily loss limit configured
- [ ] Emergency shutdown tested

## Exchange Integration
- [ ] Live API credentials configured
- [ ] API key permissions verified (trade, not withdraw)
- [ ] Rate limits within exchange limits
- [ ] Order execution latency acceptable
- [ ] Partial fill handling tested

## Capital Allocation
- [ ] Initial capital deposited
- [ ] Capital allocation per strategy defined
- [ ] Worst-case loss scenario calculated
- [ ] Loss acceptable if complete wipeout

## Monitoring
- [ ] Real-time position monitoring available
- [ ] Alert thresholds configured
- [ ] On-call rotation defined
- [ ] Dashboard for key metrics

## Legal/Compliance
- [ ] Tax implications understood
- [ ] Exchange terms of service reviewed
- [ ] Trading regulations understood

> Live trading involves real financial risk. Losses can exceed initial investment.
